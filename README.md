# Hledání nejkratší cesty v bludišti

Sandra Bělohoubková - BEL0153

Ve složce `knihovna` se nacházejí soubory s funkcemi pro hledání cesty bludištěm
a s funkcemi pro jejich generování. V notebooku `examples/examples.ipynb` je
ukázáno použití těchto funkcí. Ve složce `data` se nacházejí testovací data pro
hledání cesty a ve složce `examples/obrazky` jsou obrázky šablon a ukázky
vygenerovaných bludišť.

V obrázku je černá zeď, bílé volné místo a červená nejkratší cesta. Instalace
knihovny se provede příkazem `pip install .` v kořenovém adresáři.
